## Cultural Considerations

### Formality Level
- Use standard Brazilian Portuguese register for ACCORD and official documentation
- Use conversational Brazilian Portuguese for UI strings and chat messages
- Technical terms can mix Portuguese with English loan words (common in tech contexts)

### Addressing Users
- Use "você" (you) for standard respectful address
- Avoid overly formal "vossa senhoria" or informal "tu" unless contextually appropriate
- For Wise Authority references, use respectful but standard terminology

### Brazilian vs. European Portuguese
- This glossary uses Brazilian Portuguese conventions
- Prefer Brazilian spellings: "aplicativo" (not "aplicação"), "tela" (not "ecrã")
- Use Brazilian technical vocabulary: "mouse" not "rato", "deletar/excluir" not "apagar"

### Technical Vocabulary
- Many technical terms retain English forms: "API", "token", "pipeline", "runtime"
- Compound tech terms can be hyphenated or written as single words: "live-memory" → "memória ao vivo"
- Acronyms remain in English: DMA, LLM, GDPR, CCPA
