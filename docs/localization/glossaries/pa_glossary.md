# CIRIS Punjabi Glossary (ਪੰਜਾਬੀ / پنجابی)

This glossary defines the canonical translations for key CIRIS terms in Punjabi. All translators must use these terms consistently across ACCORD, Guide, UI, and DMA prompts.

**Note**: Punjabi is written in two scripts:
- **Gurmukhi (ਪੰਜਾਬੀ)**: Used in Indian Punjab - primary standard for this glossary
- **Shahmukhi (پنجابی)**: Used in Pakistani Punjab - Arabic-derived script, RTL

This glossary provides both scripts where practical, with Gurmukhi as the default.

## Cultural Considerations

### Formality Level
- Use respectful Punjabi for all contexts
- "ਤੁਸੀਂ" (tusīṁ - formal you) is standard for addressing users
- Avoid casual "ਤੂੰ" (tūṁ)

### Script Considerations
- **Gurmukhi**: Primary script for Indian Punjab, left-to-right
- **Shahmukhi**: Used in Pakistani Punjab, right-to-left (Arabic-derived)
- This glossary defaults to Gurmukhi for consistency
- When targeting Pakistani audiences, Shahmukhi translations should be provided

### Punjabi-Specific Considerations
- Strong oral tradition - translations should sound natural when read aloud
- Sikh cultural influence in Indian Punjab; Islamic influence in Pakistani Punjab
- Concept of "ਸੇਵਾ" (seva - selfless service) resonates with CIRIS ethics
- "ਸੱਚ" (sach - truth) is a core cultural value, aligns with Integrity
- Significant overlap with Hindi/Urdu vocabulary but distinct grammar and script

### Tonal Markers
- Gurmukhi uses tonal markers that affect meaning
- Ensure proper representation of: ੱ (addak), ੰ (tippi), ਂ (bindi)
- Font support is critical for correct display
